Requisitos do Sistema

  • Sistema Operacional: Windows 10 ou superior (64 bits)
  • Memória RAM: Mínimo de 4GB
  • Espaço em Disco: 500MB disponíveis
  • Conexão com Internet: Necessária para sincronização de dados

Instruções de Instalação

  1. Baixe o arquivo compactado (.rar)
  2. Extraia o conteúdo para uma pasta de sua preferência
  3. Execute o arquivo SEAP-PB.exe para iniciar o aplicativo
  4. Siga as instruções na tela para configurar o sistema

Nota: Para extrair o arquivo .rar, você pode precisar de um programa como WinRAR ou 7-Zip.
